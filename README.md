# 📊 Social Media Usage & Addiction Analysis Dashboard

## 🔍 Project Overview
This project explores how social media usage patterns influence student behavior, addiction levels, and academic performance.  

Using an interactive Power BI dashboard, the analysis uncovers key trends across different demographics such as gender, age, and academic level to identify high-risk usage patterns and their potential impact.

---

## 🎯 Problem Statement
With the increasing use of social media among students, it becomes important to understand:
- How usage varies across different groups  
- Which platforms dominate user attention  
- Whether excessive usage is linked to addiction and academic decline  

This project aims to transform raw data into actionable insights that can support better awareness and decision-making.

---

## 🛠️ Tools & Technologies
- Power BI (Dashboard & Visualization)
- Excel (Data Cleaning & Preparation)

---

## 📂 Dataset Description
The dataset contains information on:
- Student demographics (age, gender, academic level)
- Daily social media usage (hours)
- Platform preferences (Instagram, TikTok, WhatsApp, etc.)
- Addiction scores
- Academic performance indicators
- Lifestyle factors such as sleep and relationship status

---

## 📈 Key Insights

### 📌 1. Usage Patterns by Academic Level
- High school students show the **highest average usage hours**
- Usage decreases slightly at higher academic levels  
👉 Suggests younger users are more prone to excessive engagement

---

### 📌 2. Platform Dominance
- Platforms like **WhatsApp, TikTok, and Instagram** dominate usage
- Certain platforms show **higher correlation with addiction scores**  
👉 Indicates platform-specific behavioral impact

---

### 📌 3. Addiction vs Usage Relationship
- Higher daily usage is strongly associated with **increased addiction scores**
👉 Confirms that excessive screen time directly contributes to addictive behavior

---

### 📌 4. Academic Impact
- Increased social media usage shows a **negative trend with academic performance**
👉 Suggests potential impact on focus and productivity

---

### 📌 5. Behavioral & Lifestyle Insights
- Students with higher usage also show:
  - Lower average sleep hours  
  - Higher conflict levels in relationships  
👉 Indicates broader lifestyle impact beyond academics

---

## 📊 Dashboard Features
